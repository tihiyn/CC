# Занятие 2. Имена переменных

```
limit - reviewsEntryNum
// количество записей об отзывах
```

```
fromHead, fromTail - moveFromHeadNode, moveFromTailNode
// переменные в функции для инвертирования порядка элементов в двусвязном списке
```

```
firstPointer, secondPointer - pointerToFirstListNode, pointerToSecondListNode
// переменные указывающие на узлы перового и второго списка соответственно
```

```
map - openCloseBracketsMap
// карта, хранящая в качестве ключей открывающиеся скобки, а в качестве занчений соответствующие закрывающиеся скобки
```

```
a, b - firstOperand, secondOperand
// операнды в ариметическом выражении
```

```
s - lowerСaseStringWithoutSpaces
// строка, в которой все символы в нижем регистре и нет пробелов
```

```
w, h - windowWidth, windowHeight
// размеры окна
```

```
C - validCharatersArray
// массив допустимых значений
```

```
flag - isInputTypeCorrect
// булева переменная, означающая корректность типа ввода
```

```
A - hexNumbersMatrix
// матрица с шестнадцатеричными числами
```

```
chislo - convertedHexToBinNum
// число, переведённое из 16-ой СС в 2-ую СС
```

```
t_v - inputType
// тип ввода
```
