# Занятие 6. Имена классов

## 3.1

```
ReceiveLinksServer - AgentPathService
// класс, содержащий методы для создания маршрутов для агентов
```

```
UserManager - UserAccessCntroller
// класс для управления доступами пользователей
```

```
CustomerOrderData - CustomerOrderRepository
// класс, который позволяет управлять данными о заказах клиентов
```

```
CustomerEntity - Customer
// класс клиента
```

```
EventManager - EventHandler
// класс, отвечающий за обработку событий
```

## 3.2

```
linkOptimizer, routeOptimizer, pathOptimizer - linkOptimizer
// все эти объекты являются объектами класса LinkOptimizer, который содержит методы для оптимизации маршрутов
```

```
getCoords, calcCoords - getCoords
// метод для получения координат
```

```
getMovieById, findMovieById, receiveMovieById - getMovieById
// метод для получения фильма по его ID
```

```
executeTask, runTask - runTask
// метод, который выполняет задачи
```

```
userValidator, userChecker - userValidator
// объекты класса для валидации пользователей
```

```
getAnimals, getAllAnimals - getAnimals
// метод для получения списка всех животных
```

```
deleteAnimal, deleteAnimalById - deleteAnimal
// метод для удаления животного
```
