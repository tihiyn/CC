# Занятие 7. Имена функций/методов

```
reverseList - reverseOrder
// метод в классе LinkedList для инвертирования порядка элементов
```

```
validate - validateEmailAddress
// метод для валидации адреса электронной почты
```

```
isBalanced - isBracketsBalanced
// функция, которая проверяет, корректно ли в строке расположены скобки
```

```
compute - calcInPostfixNotation
// функция, которая выполняет вычисления арифметических выражений в постфиксной нотации
```

```
merge - mergeOrderedLists
// функция, которая объединяет два упорядоченных списка в один упорядоченный список
```

```
save - saveDocumentToTextFile
// метод, который сохраняет документ в текстовый файл
```

```
getSessionInfo - getCurrentSessionUserId
// метод для получения ID пользователя в текущей сессии
```

```
optimizationAlgorithm - buildOptimalRoute
// метод для построения оптимального маршрута
```

```
getMatrix - getReachabilityMatrix
// метод для построения матрицы достижимости графа
```

```
getTime - calculateTimeBetweenPoints
// метод, который рассчитывает время кратчайшего пути между 2 точками
```

```
prepareRequest - addHeadersToRequest
// метод, который добавляет необходимые заголовки к запросу
```

```
calcPayment - calcAnnuityPayment
// метод для рассчёта аннуитетного платежа
```
