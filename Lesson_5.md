# Занятие 5. Имена, которых следует избегать

```
result - intersectionResult
// результат пересечения множеств (тоже множество)
```

```
operationTime - operationTimeInMs
// время выполнения операции в миллимекундах
```

```
l1, l2 - firstProperValue, secondProperValue
// элементы пар, полученных в результате сочетаний всех собственных значений оператора
```

```
period - minMaxInsurancePeriod
// минимальный среди всех максимальных периодов страховок
```

```
sum - discountsSum
// сумма всех скидок
```

```
payment - averageMonthlyPayment
// средний ежемесячный платёж
```

```
Set<String> strategiesList - Set<String> strategies
// множество стратегий для рассчёта
```

```
method, methodValue - methodName, methodCalcValue
// methodName - название метода для решения СЛАУ
//methodCalcValue - значение, полученное при использовании метода
```

```
phoneNumber+7 - phoneNumWithRusCode
// номер телефона в формате +7...
```

```
buffer - imageProcessingBuffer
// временно хранилище при обработке изображений
```

```
inputData, data - inputUserData, validatedUserData
// inputUserData - данные о пользователе, которые поступили на вход
// validatedUserData - данные о пользователе после валидации
```

```
changesTemperatureTable - changesOfTemperature
// таблица с изменениями температуры
```
